Les outils

Voici les ressources nécessaires (et facultatives) pour la réalisation des projets.

  Description
Visual Studio Code Editeur de code
Chrome, Firefox Navigateur

Chrome est recommandé étant le navigateur le plus utilisé.

Les extensions à installer dans Visual Studio Code :

  • Ctrl + Shift + X pour ouvrir le marketplace
    •   Description
    Live Server Une extension permettant de lancer localement un serveur de développement. S'actualise automatiquement à chaque fois que le code est sauvegardé.
    (recommandé) Javascript (ES6) code snippets Une extension qui permet de générer du code. Par exemple, tapez "div" et l'extension vous proposera div, puis en faisant entrer il génèrera les balises.
    (recommandé) Prettier - Code Formatter L'extension permet en faisant Clic Droit -> Format Document With... de formater le code selon les normes.
    (optionel) Bracket Pair Colorizer 2 L'extension permet d'avoir des couleurs pour le code plus lisible.
