Les ressources

Voici les ressources nécessaires (et facultatives) pour la réalisation des projets.

  • Visual Studio Code, un éditeur de code.
  • Un navigateur comme Chrome ou Firefox.

Chrome est recommandé car il est le navigateur le plus utilisé (environ 64%).

Les extensions dans Visual Studio Code (Ctrl + Shift + X pour ouvrir)

  • Live Server
  • (recommandé) Javascript (ES6) code snippets
  • (recommandé) Prettier - Code Formatter
  • (optionel) Bracket Pair Colorizer 2

Live Server permet de lancer localement votre code dans votre navigateur et rafraichi la page à chaque fois que vous allez sauvegarder. Pour le lancer, lorsque vous êtes dans votre projet, cliquez sur Go Live en bas à droite de l'application.

Prettier permet en faisant un clique droit sur votre code puis Format Document With (prettier) de formater votre code javascript.
